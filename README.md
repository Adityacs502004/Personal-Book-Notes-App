# Personal-Book-Notes-App

A full-stack web application to manage books, write notes, rate them, and maintain a personal reading collection.

Built using Node.js, Express, PostgreSQL, EJS, HTML, CSS, and JavaScript.


Features :-

📖 Add new books (title + author)

✏️ Edit book details

📝 Write and update notes for each book

⭐ Interactive star rating system

🗑 Delete books (with cascading note deletion)

🗄 PostgreSQL database integration

🎨 Clean and responsive UI
